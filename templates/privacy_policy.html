1. Information We Process

We process the minimum information required to deliver model-based MRI image analysis.

  • Uploaded MRI image files submitted for inference.
  • Technical metadata such as request time, browser type, and performance diagnostics.
  • No permanent patient profile is created by default on this website.

2. How We Use Information

Processed data is used to provide predictions and maintain service quality and reliability.

Prediction Delivery

To classify brain MRI scans into supported categories with confidence outputs.

System Monitoring

To detect failures, improve response time, and prevent misuse.

Model Improvement

To improve model quality only through controlled, compliant workflows.

3. Data Retention

Unless stated otherwise in future releases, uploaded images are intended for transient processing and are not retained as long-term records.

Avoid uploading identifiable personal information in file names or image annotations unless your institutional workflow permits it.

4. Security Controls

  • Industry-standard transport security for in-transit data.
  • Role-limited access to system administration functions.
  • Routine monitoring for abnormal activity and service abuse.

5. Your Privacy Choices

You may contact us to ask questions, report concerns, or request data-handling clarification.

Contact email: support@neuroscan-ai.com
