Technical Specifications

Model Architecture

  • Type: Convolutional Neural Network (CNN)
  • Input Size: 224 × 224 × 3 (RGB images)
  • Total Layers: 28 layers
  • Convolutional Layers: 16
  • Pooling Layers: 5 (Max Pooling)
  • Dense Layers: 3 (including output)
  • Parameters: ~12.5 million trainable
  • Activation: ReLU (hidden), Softmax (output)

Training Configuration

  • Framework: TensorFlow 2.x & Keras
  • Training Samples: 7,023 MRI scans
  • Validation Samples: 1,511 MRI scans
  • Test Samples: 1,200 MRI scans
  • Epochs: 50 with early stopping
  • Batch Size: 32
  • Optimizer: Adam (lr=0.0001)
  • Loss Function: Categorical Crossentropy

Data Augmentation

  • Rotation: ±20 degrees
  • Width/Height Shift: ±10%
  • Zoom Range: 0.9 to 1.1
  • Horizontal Flip: Yes
  • Brightness Adjustment: ±15%
  • Contrast Adjustment: ±10%
  • Gaussian Noise: σ = 0.05
  • Elastic Deformation: Applied

Performance

  • Inference Time: 1.8 seconds (CPU)
  • GPU Inference: 0.3 seconds
  • Memory Usage: 450 MB
  • Model Size: 48 MB
  • Throughput: 500+ scans/hour
  • Deployment: Web, Cloud, Edge
  • Scalability: Horizontal & Vertical
  • Reliability: 99.9% uptime
