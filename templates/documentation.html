1. Workflow Overview

  1. Open the Get Diagnosis page.
  2. Upload an MRI image in supported format (PNG, JPG, JPEG).
  3. Run analysis and review class probabilities and confidence.
  4. Cross-check findings with clinical assessment before final decisions.

2. Output Format

Prediction response includes:

  • prediction: top predicted class label.
  • confidence: model confidence value (0 to 1).
  • predictions: class-wise probability distribution.

3. Supported Classes

Glioma

Tumors arising from glial cells with varied aggressiveness.

Meningioma

Tumors originating in the meninges surrounding brain and spinal cord.

Pituitary

Tumors located around the pituitary gland region.

No Tumor

No visible tumor-like pattern according to the model.

4. Model and Processing Notes

  • Transfer-learning architecture using VGG16 backbone.
  • Input images are resized to 150 x 150 and normalized.
  • Probability scores are generated via softmax output layer.

5. Responsible Use

AI output should support, not replace, the judgment of certified medical professionals.
